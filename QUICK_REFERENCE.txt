# QUICK REFERENCE CARD

## 🚀 Starting Your System

**EASIEST METHOD:**
1. Find **START.bat** in your folder
2. Double-click it
3. Wait for success message
4. Browser opens automatically
5. Go to **Admin Panel**

---

## 📊 ADMIN PANEL - What You Do

### Create Learning Point
**Where:** Learning Points tab
- **Type:** Pick Sr App / Manager App / Web App / Scenario Based
- **Module Name:** What you want to teach
- **Description:** (Optional) Details about it

### Create Training Session  
**Where:** Training Sessions tab
- **Learning Point:** Which module?
- **Date & Time:** When?
- **Add Participants:** Click button, type names

### Create Tasks
**Where:** Tasks tab
- **Session:** Which training?
- **Task Name:** What to do?
- **Description:** How to do it?
- **Deadline:** When due?
- **Assign To:** Who does it?
- **Attachment:** (Optional) Attach file

---

## 👤 LEARNER DASHBOARD - What Your Team Does

**Where:** http://localhost:3000/learner.html

1. Click their name (Mahi, Saimom, Moushumi)
2. See all their tasks
3. Click task to see details
4. Click "Update Task" to change status:
   - Pending → In Progress → Completed
5. See personal stats

---

## 📈 ANALYTICS - What You Monitor

**Where:** http://localhost:3000/analytics.html

See:
- Total sessions completed
- Total tasks completed  
- Completion rate per person
- Average days to complete
- All tasks in one table

## 🆘 Quick Fixes

| Problem | Fix |
|---------|-----|
| App won't start | Restart computer, run npm install again |
| Page not loading | Ctrl+R refresh, check Terminal shows "running" |
| Data disappeared | Delete training.db, restart |
| Can't find START.bat | It's in D:\Training Test folder |
| Names not showing | Restart browser, press F5 |

## 📝 Form Fields Guide

### Learning Point Form
- **Type** (required): Sr App, Manager App, Web App, or Scenario Based
- **Module Name** (required): What you call this training
- **Description** (optional): Extra details

### Training Session Form
- **Learning Point** (required): Which module?
- **Date & Time** (required): When is it?

### Task Form
- **Session** (required): Which training?
- **Task Name** (required): Title of task
- **Description** (optional): What to do
- **Deadline** (required): When due?
- **Assign To** (required): Who does it?
- **Attachment** (optional): Upload file

## 💡 Pro Tips

1. **Start small** - Create 1 module, 1 session, 1 task first

3. **Attach materials** - Help your team by providing files

5. **Get feedback** - Ask your team if system is easy
