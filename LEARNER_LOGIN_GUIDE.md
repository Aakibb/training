## What Happens When They Login

**Step by Step:**

1. **They visit login page**
   - See 3 big buttons (Mahi, Saimom, Moushumi)
   - Or textbox to type their name

2. **They click their name**
   - Instant redirect to dashboard
   - Their name shown at top
   - No password needed!

3. **Dashboard appears** showing:
   - Their stats (Total, Completed, Pending, In Progress)
   - All tasks assigned to them
   - Color-coded by status

4. **They click a task** to:
   - See full details
   - Read description
   - Download materials
   - Update status (Pending → In Progress → Completed)

5. **They click Logout** when done
   - Back to login page
   - Session ends
   - They can login again anytime

## Troubleshooting for Team Members

**"Page won't load"**
- Make sure the system is running (check with admin)
- Try refreshing (press F5)
- Try different browser

**"I don't see my tasks"**
- Make sure your name matches exactly (check spelling)
- Log out and login again
- Ask admin to verify task assignment

**"Task status didn't update"**
- Click "Update Task" button (just changing dropdown isn't enough)
- Wait for success message
- Refresh to see the update

**"Session keeps ending"**
- That's normal - log back in by clicking link again
- Your data is saved - tasks will be there
